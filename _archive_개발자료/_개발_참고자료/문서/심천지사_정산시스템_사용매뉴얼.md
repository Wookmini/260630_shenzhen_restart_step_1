# 심천지사 전도금 정산 자동화 시스템 사용 매뉴얼

이 매뉴얼은 심천지사 전도금 정산 담당자가 **영수증 스캔본 정리부터 데이터 자동 추출, 웹앱을 통한 검토 및 최종 엑셀 정산 파일 저장까지**의 모든 과정을 쉽고 빠르게 진행할 수 있도록 안내합니다.

---

## 📌 1단계: 영수증 준비 및 폴더 생성

가장 먼저 해야 할 일은 정산할 월의 영수증 이미지 파일들을 모아두는 것입니다.

1. `작업장소 (영수증 보관)` 폴더 안에 정산하실 월의 이름으로 **새 폴더**를 생성합니다.
   - 📁 **폴더명 예시:** `2026-06` (반드시 YYYY-MM 형식이어야 합니다.)
2. 생성한 폴더 안에 각 담당자별(예: Lin Wei Jian, 권유석 등) 이름으로 하위 폴더를 만들거나, 해당 경로에 스캔본 파일(PDF, JPG, PNG 등)들을 모아 넣습니다.
   - *팁: PDF 통짜 스캔본을 넣어도 프로그램이 알아서 페이지별로 쪼개어(PNG) 개별 영수증으로 인식합니다.*

---

## ⚙️ 2단계: 자동 판독(OCR) 및 정산 배치 실행

현업 담당자분들은 터미널(까만 창)을 다룰 필요 없이, 더블클릭 한 번으로 엑셀 자동 맵핑을 수행할 수 있습니다.

1. 바탕화면이나 프로젝트 최상위 폴더에 있는 **`정산실행.bat`** 파일을 **더블클릭**합니다.
2. 검은색 창이 뜨면서 `"작업할 정산 월을 입력하세요 (예: 2026-06):"` 라는 문구가 나옵니다.
3. 방금 영수증을 넣어둔 폴더의 월(예: `2026-06`)을 타이핑하고 `Enter`를 누릅니다.
4. 백그라운드에서 AI가 영수증의 글씨를 읽고(OCR), 계정과목을 분류하며, 은행 이체수수료 정합성을 검증합니다.
5. 작업이 끝나면 **"처리가 완료되었습니다."** 문구가 뜹니다. 아무 키나 누르시면 창이 닫힙니다.

> [!NOTE]
> 만약 1단계에서 폴더를 만들지 않고 월을 입력하셨다면, "폴더를 찾을 수 없습니다"라는 안내 메시지가 뜹니다. 이 경우 폴더 생성 후 다시 실행해 주시면 됩니다!

---

## 🖥️ 3단계: 웹앱 접속 및 데이터 검토

자동으로 만들어진 엑셀 파일을 웹상에서 띄워놓고 보면서 빠르고 편하게 수정할 수 있습니다.

1. **웹앱 서버 켜기 (최초 1회)**
   - 이미 웹 서버가 실행 중이라면 이 과정은 건너뛰셔도 됩니다.
   - 시스템 관리자나 담당 실무자가 **`웹앱서버실행.bat`**을 더블클릭하여 서버를 켜두어야 합니다.
2. **웹 브라우저 접속**
   - 크롬(Chrome)이나 엣지(Edge) 브라우저를 열고 주소창에 `http://localhost:8080` (또는 지정된 내부망 IP 주소)를 입력합니다.
3. 상단의 **[날짜 선택 (예: 2026-06)]** 드롭다운 버튼을 눌러 작업하신 달을 선택합니다.

---

## 🔍 4단계: 영수증 원본 대조 및 내용 수정

웹앱에는 왼쪽에는 전체 '정산 내역(테이블)'이 보이고, 특정 내역을 클릭하면 영수증 원본과 함께 값을 수정할 수 있는 '데이터 검토' 화면으로 넘어갑니다.

### 💡 주요 편의 기능 (데이터 검토 탭)
*   **영수증 확대/축소:** 영수증 원본 이미지에 마우스를 올리고 **휠(Wheel)을 위아래로 굴리면** 영수증이 확대/축소됩니다.
*   **영수증 이동(Pan):** 이미지를 확대한 상태에서 마우스 왼쪽 버튼을 **클릭한 채로 잡아끌면(Drag)** 영수증의 다른 부분으로 요리조리 시점을 이동할 수 있습니다.
*   **이전/다음 버튼:** 상단의 `< 이전` / `다음 >` 버튼을 눌러가며 1번 영수증부터 25번 영수증까지 순차적으로 훑어보실 수 있습니다.
*   **목록에서 빠르게 이동:** 화면 좌측의 '썸네일(미리보기) 사이드바'에서 특정 번호의 영수증을 클릭하면 해당 영수증으로 바로 점프합니다!

### ✍️ 데이터 수정 및 비고 작성
잘못 판독된 글씨(금액, 내역 등)가 있다면 화면 우측의 입력칸에서 바로 텍스트를 수정하세요.
추가로 기재하고 싶은 내용은 **'비고'** 란에 적어주시고 하단의 **[저장]** 버튼을 누르시면 반영됩니다.

---

## 💾 5단계: 최종 엑셀 파일 저장

웹앱에서 모든 영수증을 다 훑어보셨나요?

1. 화면 우측 상단의 **[엑셀 최종 저장]** 버튼을 누릅니다.
2. *"현재 작업하신 내용으로 [정산내역_2026-06.xlsx] 파일을 덮어쓰시겠습니까?"* 라는 경고창이 뜨면 **[확인]**을 누릅니다.
3. 엑셀 파일이 업데이트 되었습니다! 
4. 이제 `작업장소 (영수증 보관)/2026-06/` 폴더에 들어가보시면 **`정산내역_2026-06.xlsx`** 파일이 가장 최신 상태로 저장되어 있습니다.

> [!TIP]
> **엑셀 파일 열어보기**
> 엑셀 파일을 열어보시면, 웹앱에서 수기로 적으신 '비고' 내용은 **R열**에, 파표 회사코드 불일치나 은행 이체수수료 불일치 등 시스템이 자동으로 찾아낸 위험요소/경고 사항은 **S열(항목)**에 예쁘게 분류되어 나타납니다!
