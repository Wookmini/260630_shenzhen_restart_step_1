심천지사 전도금 정산 시스템 매뉴얼

영수증 스캔본 정리부터 데이터 자동 추출, 웹앱을 통한 검토 및 최종 엑셀 정산 파일 저장까지의 과정을 안내합니다.

1

영수증 준비 및 폴더 생성

가장 먼저 해야 할 일은 정산할 월의 영수증 이미지 파일들을 모아두는 것입니다.

  1. 작업장소 (영수증 보관) 폴더 안에 정산하실 월의 이름으로 새 폴더를 생성합니다.
    • 폴더명 예시: 2026-06 (반드시 YYYY-MM 형식이어야 합니다.)
  2. 생성한 폴더 안에 각 담당자별(예: Lin Wei Jian, 권유석 등) 이름으로 하위 폴더를 만들거나, 해당 경로에 스캔본 파일(PDF, JPG, PNG 등)들을 모아 넣습니다.
💡

통짜 PDF 스캔본도 괜찮습니다!
PDF 스캔본 하나를 통째로 넣어도 프로그램이 알아서 페이지별로 쪼개어(PNG) 개별 영수증으로 인식합니다.

2

자동 판독(OCR) 및 정산 실행

복잡한 터미널 조작 없이 더블클릭 한 번으로 엑셀 자동 맵핑을 수행할 수 있습니다.

🚨

[필수 규칙] 이전 달 엑셀 파일 저장 (수식 캐시 확정)

새로운 달의 배치를 돌리기 직전에, 반드시 이전 달의 엑셀 파일을 열어 저장(Ctrl+S) 후 닫아주셔야 합니다.
파이썬은 엑셀 고유의 복잡한 수식(환율, 입출금 가감)을 스스로 연산하지 못합니다. 사용자가 엑셀로 직접 열어 저장을 눌러주어야만 엑셀 엔진이 이월 잔액을 메모리에 확정(Cache)시켜 다음 달 기초 잔액으로 정확히 넘겨줄 수 있습니다.
(예: 7월 정산을 돌리기 전, 6월 폴더의 2026-06.xlsx 파일을 열고 저장(Ctrl+S) 후 닫기)

  1. 바탕화면이나 프로젝트 최상위 폴더에 있는 정산실행.bat 파일을 더블클릭합니다.
  2. 검은색 창이 뜨면서 "작업할 정산 월을 입력하세요 (예: 2026-06):" 라는 문구가 나옵니다.
  3. 방금 영수증을 넣어둔 폴더의 월(예: 2026-06)을 타이핑하고 Enter를 누릅니다.
  4. 백그라운드에서 AI가 영수증의 글씨를 읽고(OCR), 계정과목을 분류하며, 은행 이체수수료 정합성을 검증합니다.
  5. 작업이 끝나면 "처리가 완료되었습니다." 문구가 뜹니다. 아무 키나 누르시면 창이 닫힙니다.
⚠️

안내 메시지 확인: 만약 1단계에서 폴더를 만들지 않고 월을 입력하셨다면, "폴더를 찾을 수 없습니다"라는 한글 안내 메시지가 뜹니다. 이 경우 폴더 생성 후 다시 실행해 주시면 됩니다!

🚨

[중요 주의사항] 영수증 사진을 교체하거나 순서를 바꿀 때!
평소 엑셀 내용을 조금 수정하거나 영수증을 단순히 추가하실 때는 엑셀 파일을 절대 지우시면 안 됩니다(수동 작성 내용 보존).
다만, 특정 번호의 영수증 사진을 아예 다른 사진으로 교체(덮어쓰기)하거나 순서를 대대적으로 바꿀 경우에는, 기존에 생성되어 있던 심천지사 전도금 정산 양식_...xlsx 엑셀 파일을 반드시 먼저 삭제한 뒤 정산실행.bat을 돌려주세요. 삭제하지 않으면 사용자가 이전에 수정한 과거 기록이 새로운 영수증에 잘못 덮어씌워져 금액이 꼬일 수 있습니다.

3

웹앱 접속 및 데이터 검토

자동으로 만들어진 엑셀 파일을 웹상에서 띄워놓고 보면서 빠르고 편하게 수정할 수 있습니다.

  1. 웹앱 서버 켜기 (최초 1회)
    • (이미 웹 서버가 켜져 있다면 이 과정은 생략합니다.) 관리자가 웹앱서버실행.bat을 더블클릭하여 실행해 두어야 합니다.
  2. 웹 브라우저 접속
    • 크롬(Chrome)이나 엣지(Edge)를 열고 주소창에 http://localhost:8080 을 입력합니다.
  3. 상단의 날짜 선택 드롭다운 버튼을 눌러 방금 작업하신 달을 선택합니다.
4

영수증 원본 대조 및 내용 수정

정산 내역 테이블에서 특정 내역을 클릭하면 영수증 원본과 함께 값을 수정할 수 있는 화면으로 넘어갑니다.

💡 주요 편의 기능 (데이터 검토 탭)

  • 영수증 확대/축소: 영수증 이미지에 마우스를 올리고 마우스 휠(Wheel)을 위아래로 굴려보세요.
  • 영수증 이동(Pan): 이미지를 확대한 상태에서 마우스 왼쪽 버튼을 클릭한 채로 잡아끌면(Drag) 이리저리 이동할 수 있습니다.
  • 이전/다음 버튼: 상단의 < 이전 / 다음 > 버튼을 눌러가며 영수증을 순차적으로 빠르게 훑어볼 수 있습니다.
  • 목록에서 점프: 좌측의 '썸네일 사이드바'에서 특정 번호의 영수증을 클릭하면 해당 영수증으로 바로 이동합니다.

✍️ 데이터 수정 및 비고 작성

  • 잘못 판독된 글씨(금액, 내역 등)가 있다면 화면 우측의 입력칸에서 바로 텍스트를 수정하세요.
  • 추가로 기재하고 싶은 내용은 '비고' 란에 적어주시고 하단의 저장 버튼을 누르시면 즉시 엑셀 데이터에 반영됩니다.
5

최종 엑셀 파일 저장

웹앱에서 모든 영수증 검토가 끝났다면 최종적으로 엑셀 파일에 저장해야 합니다.

  1. 화면 우측 상단의 엑셀 최종 저장 버튼을 누릅니다.
  2. "현재 작업하신 내용으로 파일을 덮어쓰시겠습니까?" 라는 알림창이 뜨면 확인을 누릅니다.
  3. 이제 작업장소 (영수증 보관)/2026-06/ 폴더에 들어가보시면 정산내역_2026-06.xlsx 파일이 가장 최신 상태로 업데이트되어 있습니다!
✨

엑셀 파일 최종 결과물 확인 (최신 시스템 반영)
- 스마트 환율 및 이월 잔액 계산: 이전 달의 이월 잔액(RMB)이 모두 소진될 때까지는 전월 최종 환산요율이, 소진 이후 건부터는 당월 최종 환산요율이 엑셀에 자동으로 일괄 적용됩니다. (전도금 수수료인 USD 10/25 항목 역시 전월 최종 요율로 계산됩니다.)
- 깔끔한 분할 표기: 이월 잔액 부족으로 정산이 2줄로 나뉘는 영수증의 경우 (분할 1/2) 등의 표기가 내역(E열)을 해치지 않고 R열(비고)에 정리됩니다.
- 오류 검출 및 수동 입력 보존: 파표 회사코드 불일치나 은행 수수료 금액 불일치 등은 S열(검증)에 붉은색으로 강조되며, 웹앱에서 수동으로 입력하신 비고 사항은 기존 내용이 날아가지 않고 R열에 안전하게 누적 병합됩니다!
