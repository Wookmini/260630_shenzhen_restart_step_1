# 심천지사 영수증 정산 자동화 시스템 (SROT - Batch Edition)

본 프로젝트는 중국 심천지사에서 발생하는 증치세 영수증(VAT Invoice) 및 기타 증빙 서류를 AI 기반으로 자동 인식(OCR)하고, 월별 정산 엑셀 양식으로 일괄 변환해 주는 사내 자동화 시스템입니다.

## 🌟 주요 특징 (v2.0 업데이트)
기존의 "웹에서 한 장씩 업로드"하던 방식에서 **"폴더 기반 대량 일괄 정산(Batch Processing)"** 방식으로 아키텍처가 전면 개편되었습니다.

1. **초고정밀 AI OCR 탑재:** 무거운 GPU 환경 없이 로컬 CPU만으로도 작동하는 가벼운 AI 모델(`RapidOCR-ONNX`)을 탑재하여 찌그러짐, 90도 회전, 워터마크가 있는 악조건 속에서도 100%에 가까운 중국어 한자 인식률을 보여줍니다.
2. **자동 번역 매핑:** 잦은 결제처(예: 디디추싱, 특정 호텔 등)나 내역이 감지되면 자동으로 한글 번역을 병기해 줍니다. (사전 매핑 방식)
3. **증치세 전용 파싱:** `*카테고리*품명` 형태의 중국 증치세 영수증 규격을 완벽하게 분석하여 엉뚱한 표 헤더를 무시하고 정확한 내역을 뽑아냅니다.
4. **증빙번호 자동 정렬:** 취합된 영수증 이미지들은 엑셀에 기재되는 '증빙번호' 순서에 맞게 `001.png`, `002.png` 와 같이 깔끔하게 자동 리네이밍(Renaming) 됩니다.

---

## 🗂️ 1. 폴더 구조 및 준비 방법

이 시스템은 `영수증 보관소` 라는 기준 폴더를 사용하여 작동합니다.
매월 정산 시기가 오면 아래와 같은 폴더 구조를 만들어 영수증 사진을 넣어주세요.

```text
📂 프로젝트 폴더 (260630_shenzhen_restart_step_1)
 ┗ 📂 영수증 보관소
    ┗ 📂 2026-06               <-- 분석하고자 하는 '연도-월' 폴더
       ┣ 📂 홍길동             <-- '담당자 이름'으로 된 하위 폴더 생성
       ┃  ┣ 📝 영수증1.jpg     <-- 이름 규칙 상관없이 사진 넣기
       ┃  ┗ 📝 찰칵2.png
       ┗ 📂 이몽룡
          ┗ 📝 img_03.jpg
```
* **직원 가이드:** 직원들은 본인 이름의 폴더를 만들고, 그 달에 쓴 영수증 사진들을 파일명 상관없이 몰아넣기만 하면 됩니다.

## 🚀 2. 시스템 실행 방법 (관리자용)

모든 직원이 영수증 제출을 완료했다면, 정산 관리자는 터미널(명령 프롬프트)을 열고 아래 명령어를 실행합니다.

```bash
# 기본 사용법
python batch_processor.py [분석할 YYYY-MM]

# 실행 예시 (2026년 6월 정산 실행)
python batch_processor.py 2026-06
```

### ⚙️ 실행 시 일어나는 마법들:
1. 시스템이 `2026-06` 폴더 내부의 담당자 폴더들을 싹 순회합니다.
2. 모든 영수증 이미지의 이름을 **001, 002, 003... 순서대로 자동 변경**합니다. (이미지명 = 증빙번호)
3. **AI 모델이 이미지를 분석**하여 날짜, 금액, 내역, 판매자, 영수증 유형을 추출합니다.
4. 분석 결과가 `영수증 보관소/2026-06/` 폴더 내부에 **`정산내역_2026-06.xlsx`** 엑셀 파일로 자동 생성됩니다! (담당자란에 폴더 이름이 자동 기재됨)

## 💻 3. 차후 도입 예정 사항 (웹 대시보드)
현재는 파이썬 스크립트 실행을 통한 엑셀 다이렉트 변환(1차 가공)에 초점이 맞춰져 있습니다. 
차후에는 정산 최종 관리자가 `index.html` 형태의 대시보드를 열어, `batch_processor.py`가 추출해 놓은 원시 데이터(`data.json`)와 001.png 이미지들을 웹 화면에 띄워놓고 클릭 몇 번으로 편하게 내역을 보완하거나 수정할 수 있는 **Viewer & Editor** 환경이 통합될 예정입니다.
